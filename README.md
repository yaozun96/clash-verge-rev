## 🧩 软件简介

**Clash Verge Rev** 是基于 **Rust + Tauri 2** 框架开发的跨平台代理客户端，  
提供极致性能与现代化界面体验。

### ✨ 主要特性
- 🚀 **高性能内核**：基于性能强劲的 Rust 与 Tauri 2 框架构建  
- 🧠 **内置 Clash.Meta (mihomo)** 内核，支持切换 Alpha 版本内核  
- 🎨 **现代化 UI**：简洁美观，支持自定义主题色、代理组图标、托盘图标与 CSS Injection  
- ⚙️ **增强配置管理**：支持配置文件 Merge / Script，带语法高亮与智能提示  
- 🌐 **系统代理与守卫**：支持 TUN 模式（虚拟网卡）  
- 🧭 **可视化节点与规则编辑**  
- ☁️ **WebDAV 备份与同步**，轻松实现多设备同步配置

## ❓ 常见问题（FAQ）

**Q：Clash Verge Rev 出现闪退怎么办？**  
A：建议删除旧配置并重装最新版。

**Q：如何设置 Clash Verge Rev 全局扩展覆盖配置？**  
A：在设置 → 系统代理中启用「全局模式」。

**Q：DNS 覆盖 clash verge rev 无效？**  
A：请关闭系统代理后重启 Clash Verge Rev，再导入配置文件即可。

**Q：Clash Verge Rev 是否存在安全漏洞？**
